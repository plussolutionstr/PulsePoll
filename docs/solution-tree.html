Notlar
Shared Logic
API, Worker ve Admin aynı Application servislerini kullanır. Logic bir yerde yazar, üçü de alır.
Mobile Bağımsız
MAUI sadece HTTP ile konuşur. Domain/Application referansı yok, tamamen ayrı.
Tema Desteği
4 ResourceDictionary. Kullanıcı seçimi UserService'te saklanır, uygulama açılışında yüklenir.
