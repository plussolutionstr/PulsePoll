# Mobil Uygulama İyileştirme Listesi

## Yüksek Etki
- [ ] **Pull-to-refresh** — Anasayfa, cüzdan, anketler gibi listelerde aşağı çekerek yenileme
- [ ] **Haptic feedback** — Buton tıklamalarında, coach mark geçişlerinde, başarılı işlemlerde titreşim
- [ ] **Dark mode** — Karanlık tema desteği

## Orta Etki
- [ ] **Rate app prompt** — Anket tamamlayıp ödül aldıktan sonra uygulama değerlendirme isteği
- [ ] **Force update mekanizması** — API'den minimum versiyon kontrolü, eski versiyonları güncellemeye yönlendirme
- [ ] **Biometric auth** — Face ID / parmak izi ile giriş

## Düşük Etki
- [ ] **Micro-animasyonlar** — Bakiye değişiminde sayı animasyonu, başarılı işlemde confetti/checkmark
- [ ] **Offline cache** — İnternet yokken son verilerle açılma, çevrimdışı banner gösterme
